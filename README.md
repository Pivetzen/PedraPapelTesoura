## 🚀 Sobre o Projeto

Este projeto foi criado para demonstrar a aplicação de lógica de programação em **JavaScript** aliada a uma interface estilizada com **CSS Moderno**. 

### ✨ Funcionalidades
* **Placar em tempo real:** Acompanhe suas vitórias contra a CPU.
* **Design Responsivo:** Jogue no PC ou no Celular.
* **Feedback Visual:** Cores dinâmicas para vitória, derrota e empate.
* **Efeitos de Hover:** Botões interativos que reagem ao mouse.

---

## 🎮 Como Jogar?

Não precisa instalar nada! Basta acessar o link do projeto (via GitHub Pages) e escolher sua jogada:

1.  ✊ **Pedra** ganha de ✌ **Tesoura**.
2.  ✋ **Papel** ganha de ✊ **Pedra**.
3.  ✌ **Tesoura** ganha de ✋ **Papel**.

---

## 🛠️ Tecnologias Utilizadas

* **HTML5:** Estruturação semântica.
* **CSS3:** Layout com Flexbox, gradientes e animações de transição.
* **JavaScript (ES6):** Manipulação de DOM e lógica de aleatoriedade para a CPU.
